    About Chicken Spaceship

    Chicken Spaceship is a retro-style space shooter built entirely with HTML, CSS, and vanilla JavaScript using the native <dialog> element.

    🎮 Game Features

    • Move with arrow keys
    • Customizable fire key
    • Lives and scoring system
    • Victory and game-over conditions
    • High score saving per user - reset when another user login

    🔊 Sound & Music

    • Ambient and battle music
    • Sound effects for shots, actions and game states
    • Global mute toggle

    ⚙️ Technical Details

    • Canvas-based rendering engine
    • Compressed images and audio for fast loading
    • May load slowly on first load due to GitHub hosting
